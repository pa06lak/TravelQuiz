GET Destinations Based on IDs

Retrieve destination details based on the provided IDs.

Endpoint

GET /api/destinations

Request

  • Method: GET
  • URL: http://localhost:3000/api/destinations
  • Query Parameters:
    • ids (string, required): Comma-separated IDs of the destinations to retrieve.

Response

[
  {
    "id": "number",
    "name": "string",
    "continent": "string",
    "budget": "string",
    "duration": "string",
    "popularity": "number",
    "description": "string",
    "image": "string",
    "rating": "number",
    "count": "number"
  }
]

POST Update Destination Ratings

Update the rating for a specific destination.

Endpoint

POST /update-rating

Request Body

{
  "destinationName": "Paris",
  "userRating": 4.5
}

Response

{
  "message": "Rating updated successfully",
  "destination": {
    "id": "number",
    "name": "string",
    "rating": "number",
    "count": "number"
  }
}

POST Update Count of Question Options

Endpoint

POST /api/update-count

Request Body

}

Response

{
